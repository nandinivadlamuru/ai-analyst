# Operations Playbook Notes

The analytics operations team runs a weekly triage every Monday at 09:00.
Each region must appoint one data owner responsible for source-system completeness checks.

Escalation rules:
- If incidents in a region exceed 12 in a quarter, assign a remediation lead.
- If turnaround days remain above 7 for two consecutive quarters, trigger process redesign.

North region currently uses automated validation on ingestion; West region is migrating to the same pipeline.
